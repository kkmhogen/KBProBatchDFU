# KBProBatchDFU 
## Requirements
Development environment:  
Android 5.1  
minSdkVersion 21

## Installation

KBProBatchDFU is used to batch upgrade KKM's KBeacon device. This app can automatically connect KBeacon devices in batches for firmware upgrades.
